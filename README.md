# ShpoolkenDB

База данных филаментов для 3D-печати, созданная потому что [SpoolmanDB](https://github.com/donkie/SpoolmanDB) решил, что поддержка — это опционально.

## Суть проблемы

Оригинальный SpoolmanDB существует с 2022 года. За это время:

- PR с новыми филаментами ждут рассмотрения по полгода
- На issue отвечают... ну, отвечают когда-нибудь
- Свежие цвета Bambu, Prusa, Polymaker — где-то там, в очереди
- Список производителей выглядит так, будто время остановилось в 2023-м

При этом Spoolman (программа) — отличный проект. Но база данных... база данных грустит.

## Что мы сделали

Мы взяли существующую базу, выкинули пыль и начали её реально поддерживать. Новые филаменты? Добавляем. Новые цвета? Добавляем. Issue? Отвечаем. PR? Рассматриваем за разумное время.

Революция? Нет. Просто работающая база данных.

Живой сайт: [https://dontneedfriends-jpg.github.io/ShpoolkenDB/](https://dontneedfriends-jpg.github.io/ShpoolkenDB/)

## Структура репозитория

```
filaments/           # JSON-файлы по производителям
materials.json      # Свойства материалов (PLA, PETG, ABS...)
scripts/             # Скрипты компиляции
app/                 # Веб-интерфейс (React + Tailwind)
```

При деплое `filaments/` + `materials.json` компилируются в единый `filaments.json`, который раздаётся через GitHub Pages.

## Хочешь добавить филамент?

1. Форкни репозиторий
2. Найди файл производителя в `filaments/` (или создай новый)
3. Отправь PR
4. Получи ответ. Не через год, а в адекватные сроки.

Или воспользуйся формой на сайте — она сгенерирует готовый Issue на GitHub.

## Поля в JSON

| Поле | Описание |
|------|----------|
| `name` | Название. Используй `{color_name}` для автоподстановки |
| `material` | Материал (PLA, PETG, ABS...) |
| `density` | Плотность в г/см³ |
| `weights` | Массив весов: `{weight, spool_weight, spool_type}` |
| `diameters` | Массив диаметров в мм |
| `colors` | Массив цветов: `{name, hex}` |
| `extruder_temp` | Температура экструдера (°C) |
| `bed_temp` | Температура стола (°C) |
| `finish` | "matte" или "glossy" |
| `pattern` | "marble" или "sparkle" |
| `translucent` | Прозрачный |
| `glow` | Светится в темноте |
